MA : Web API Attribute Table

KEY VALUE DESCRIPTION VALUE TYPE
user-id Every user will have an individual account which will require their user-id. String
sex Specification if a person comes under male or female category, as there are slight differences in effects of health disease in different sexes. 0 = Female, 1 = Male. Integer
age age plays a major role when it comes to health, therefore specification of health in years should be provided Integer
education 1 = Some High School; 2 = High School or GED; 3 = Some College or Vocational School; 4 = college. Integer
currentSmoker Whether or not the person smokes at present, it defines their behavioural health issues. Integer
cigsPerDay Average cigarettes a person smokes per day Integer
BPMeds Whether or not the person is on blood pressure medication. 0 = Not on Blood Pressure medications; 1 = Is on Blood Pressure medications. Integer
prevalentStroke Whether or not the person previously had a stroke. 0 = no, 1 = yes. Integer
prevalentHyp Whether or not the person was hypertensive. 0 = no, 1 = yes. Integer
diabetes Whether or not the person has diabetes. 0 = no, 1 = yes. Integer
sysBP Value of person's systolic blood pressure. Value has to be provided in mmHg. Float
diaBP Value of person's diastolic blood pressure. Value has to be provided in mmHg. Float
BMI Body Mass Index of the person. Body Mass Index calculated as: Weight (kg) / Height(meter-squared). Float
heartRate In medical research, variables such as heart rate though in fact discrete, yet are considered continuous because of large number of possible values. Value has to be provided in Beats/Min (Ventricular). Integer
glucose Value of person's Glucose level. Value has to be provided in mg/dL. Float
